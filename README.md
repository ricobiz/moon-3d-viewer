# AI Trading Bot — MetaTrader 5

Профессиональный AI-торговый бот с современным интерфейсом для MetaTrader 5. Описывайте свои стратегии на обычном языке — AI автоматически генерирует готовый MQL5 код.

## Возможности

- **AI Генерация Стратегий** — Описывайте стратегию на русском или английском, получайте готовый MQL5 код
- **Live Trading** — Подключение к MT5 через Python-мост для реальной торговли
- **Мониторинг позиций** — Все открытые сделки в реальном времени
- **Аналитика** — Equity curve, P&L по месяцам, Win Rate и другие метрики
- **Поддержка всех инструментов** — Forex, Крипта, Металлы, Индексы
- **OpenRouter API** — Подключайте любую AI модель (Claude, GPT-4, Gemini, и др.)
- **Управление рисками** — Фиксированный лот, % от баланса, дневной лимит убытков

## Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск приложения

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

### 3. Настройка

1. Перейдите в **Settings**
2. Введите API ключ [OpenRouter](https://openrouter.ai/keys)
3. Выберите AI модель (рекомендуется Claude 3.5 Sonnet)

### 4. Генерация стратегии

1. Перейдите в **AI Strategies → New Strategy**
2. Опишите вашу стратегию обычным языком
3. Нажмите **Generate MQL5 Strategy**
4. Скачайте `.mq5` файл и установите в MetaTrader 5

## Подключение к MetaTrader 5

Для реальной торговли нужен Python-мост (работает только на Windows с установленным MT5).

### На Windows:

```bash
cd mt5-bridge
pip install -r requirements.txt
python bridge.py
```

Или запустите `start.bat`.

Затем в настройках приложения:
- Включите **MT5 Bridge**
- Укажите URL: `http://localhost:8765` (или IP вашего ПК)
- Нажмите **Test Connection**

## Установка стратегии в MT5

1. Скачайте `.mq5` файл из приложения
2. В MetaTrader 5: `File → Open Data Folder → MQL5 → Experts`
3. Скопируйте файл в папку `Experts`
4. В MT5 нажмите `F5` для компиляции
5. Перетащите EA на график
6. Включите **Auto Trading**

## Технический стек

- **Frontend**: Next.js 14, React, Tailwind CSS
- **Charts**: Recharts
- **State**: Zustand (с localStorage)
- **AI**: OpenRouter API (Claude, GPT-4, Gemini, и др.)
- **MT5 Bridge**: Python + Flask + MetaTrader5 library

## Примеры стратегий

### RSI + MACD
```
Торгуй EURUSD на M15. Покупай когда RSI (14) опускается ниже 30 и
возвращается обратно, И MACD линия пересекает сигнальную снизу вверх.
Стоп-лосс 30 пунктов, тейк-профит 60 пунктов.
```

### Пересечение MA
```
Торгуй BTCUSDT на H1. Покупай когда EMA20 пересекает EMA50 снизу вверх.
Закрывай позицию при обратном пересечении. ATR стоп-лосс 1.5x.
Риск 1% от депозита. Фильтр тренда: EMA200.
```

### Bollinger Bands
```
Торгуй XAUUSD на H4. Покупай при пробое верхней полосы Боллинджера
(20, 2.0). Стоп-лосс под средней линией. Соотношение риска к прибыли 1:2.
Максимум 3 открытые позиции. Трейлинг стоп 20 пунктов.
```
